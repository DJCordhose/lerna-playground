# lerna-playground
Playground for a lerna mono repo

## Steps

1. `npx lerna init`
  * https://github.com/lerna/lerna#getting-started
1. yarn workspaces
  * https://classic.yarnpkg.com/lang/en/docs/workspaces/
1. `yarn add typescript -D`  
1. `npx tsc --init`
1. https://aka.ms/tsconfig.json
